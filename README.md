# Dialogflow Webhook

This project is a webhook for Dialogflow that helps users find and reserve books. The webhook can handle book search queries, check availability, and confirm reservations.

## Features
- Search for books by author.
- Check if a book is available for reservation.
- Confirm or decline reservations.

## Prerequisites
- Node.js 14.x or higher
- Express.js
- Body-parser
